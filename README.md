# paimon
鸿蒙5.0和6.0上的米哈游旗下云游戏启动器， 目前支持云原神和云崩坏·星穹铁道

### 支持游戏
- [x] 云原神
- [x] 云崩坏·星穹铁道
- [ ] 云绝区零: 不支持, 因为没有网页版
